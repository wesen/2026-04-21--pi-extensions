# Improve shared extension launcher UX

## Overview

This ticket captures the design work for improving the shared Pi extension framework UX. The requested changes focus on `/px` launcher continuity, fuzzy filtering, list wraparound, scrollable details/help panes, taller overlays, and a prompto shortcut/insertion workflow that does not depend on typing `/prompto` at the beginning of the editor.

The primary deliverable is an intern-ready implementation guide with line-anchored evidence and phased implementation instructions.
